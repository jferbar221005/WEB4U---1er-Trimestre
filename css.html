CSS Colors

The `color` property specifies the text color of an element.

p {
  color: blue;
}

Result:

Learn more on the W3C Documentation.

CSS Margins

The `margin` property defines the space around an element, outside its border.

div {
  margin: 20px;
}

Result:

Learn more on the W3C Documentation.

CSS Font Family

The `font-family` property sets the font of an element's text.

h1 {
  font-family: Arial, sans-serif;
}

Result:

Learn more on the W3C Documentation.

CSS Background Colors

The `background-color` property sets the background color of an element.

div {
  background-color: rgb(255, 0, 0);
}

Result:

Learn more on the W3C Documentation.

CSS Borders

The `border` property sets the style, width, and color of an element's border.

div {
  border: 2px solid black;
}

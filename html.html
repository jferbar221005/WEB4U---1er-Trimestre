HTML Headings

The <h1> to <h6> tags define headings, which are used to structure content hierarchically.

<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Sub-subheading</h3>

Learn more on the W3C Documentation.

HTML Links

The <a> tag creates hyperlinks that connect pages or resources.

<a href="https://example.com">Visit Example</a>

Learn more on the W3C Documentation.

HTML Paragraph

The <p> tag is used to define paragraphs in HTML, which organize text into blocks.

<p>This is a paragraph of text.</p>

HTML Images

The <img> tag is used to embed images in a webpage.

HTML Lists

HTML supports both ordered (<ol>) and unordered (<ul>) lists, as well as description lists (<dl>).

<!-- Unordered List -->
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

<!-- Ordered List -->
<ol>
  <li>Step 1</li>
  <li>Step 2</li>
  <li>Step 3</li>
</ol>
